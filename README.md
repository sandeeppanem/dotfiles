  My Dotfiles - Just the way I like it!

###Pre-requisites

```bash
brew install ctags coreutils macvim ack python fasd irssi
```


###How to Install

```bash
git clone https://github.com/subszero/dotfiles.git dotfiles
cd dotfiles
git submodule init
git submodule update
dotfiles/install.sh
```
